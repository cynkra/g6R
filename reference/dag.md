# Example DAG graph

The graph is a directed acyclic graph

## Usage

``` r
data(dag)
```

## Format

A list with 2 data frames:

- nodes:

  data frame with 12 rows for the nodes.

- edges:

  data frame with 12 rows for the edges.

- combo:

  data frame with 3 rows for the combos.
